## 🚀 Now

Senior Backend Engineer at Labgenomics (IT team). I design, build, and operate the company's core backend — instrument-integrated LIMS, LIS–ERP revenue integration, and an on-prem → AWS migration. I look first at where data accumulates and how it flows, design state and consistency explicitly, and trace the cause rather than the symptom.
